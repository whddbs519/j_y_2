1.시리얼 통신이란?

시리얼 통신은 아두이노와 컴퓨터간의 통신을 뜻한다. 여기서 시리얼 통신은 직렬 통신을 말하는 뜻이다. 예를 들어 101102 와 같은 데이터를 PC를 통해 아두이노로 보낼 때 101102의 데이터가 한번에 다 같이 통신되지 않고 1 0 1 1 … 과 같이 하나의 데이터씩 통신을 하게 된다. 이때 사용하는 것이 시리얼 모니터이다. 아래의 그림에서 빨간색으로 네모 표시가 된 곳이 시리얼 모니터로 들어가는 버튼이다.

2.시리얼 모니터

시리얼 모니터는 아두이노와 PC간의 대화창이라고 생각하면 쉽다. 시리얼 모니터를 통해 아두이노와 대화(명령)를 할 수 있으며 아두이노의 진행상황도 쉽게 볼 수 있다. 또한 시리얼 모니터를 통해 디버깅도 할 수 있다. ( 디버깅 : 버그를 잡는 행위 ) 시리얼 통신을 할 때 아두이노의 디지털핀 0번핀과 1번핀은 사용할 수 없다.

3.시리얼 통신 함수

① Serial.begin( 통신속도 ); : 시리얼 통신을 시작하겠다는 의미이다. ()에는 통신 속도가 들어간다. Serial.begin(9600); //9600의 속도로 시리얼 통신을 시작하겠다.

② Serial.print(“출력값”); :”출력값”을 시리얼 모니터를 통해 출력하겠다. Serial.print(“hello”); //hello라는 출력값을 시리얼 모니터에 출력하겠다. Serial.println(“world”); //println함수는 ln이 붙어지면서 줄 바꿈이 일어남. 
	#include <studio.h>
void setup() {
  Serial.begin(9600);
}

void loop() {
  Serial.print("hello world");
}
③ Serial.available(); :시리얼 통신으로 수신된, 읽지 않은 상태의 데이터수를 반환한다. (남아 있지 않으면 0)
④ Serial.read(); :시리얼 통신 수신 버퍼에서 첫번째 문자를 읽어 반환하다. 수신 버퍼가 비어 있으면 -1을 반환한다. Serial.println(Serial.read());
