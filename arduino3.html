1.자료형이란?

:유효한 데이터 크기와 여러 종류의 데이터(정수,실수,문자)등을 식별 가능하게 구분

2.자료형의 종류

-논리형 : Boolean (참과 거짓)

-정수형 : int (1,2,3 ..)
-문자형 : char (a,b,c ..)
-실수형 : double ( 1.2 ,3.5 ..)

3.변수

:프로그램에서 데이터 값들을 일시적으로 저장하는 공간을 말한다.

4.변수 선언 방법

:자료형 변수명 = 해당값
변수명은 자신이 지정해주는 이름
아래의 코드 두개는 서로 같은 코드를 두가지 방식으로 표현한 방법 

	#include <studio.h>

	int buttonState;
	buttonState = 20;

    int buttonState = 20;

5.상수

: 데이터 값이 변하지 않는 고정된 값 ( 예시 : pi = 3.141592….)
변수의 선언 방법과 같으나 앞에 const라는 단어를 붙여줌

	#include <studio.h>

	const int pi = 3.141592;

6.자주 사용되는 추가적인 함수

pinMode(핀번호,입력(or출력))
digitalWrite(핀번호,상태)

	#include <studio.h>

	pinMode(13,OUTPUT);  //13번 핀을 출력으로 사용하겠다.
	digitalWrite(13,HIGH);	//13번핀을 HIHG(5V)상태로 만들겠다.

7.실습
